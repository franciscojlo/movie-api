1. Get All Movies

URL: GET /movies

Description: Return a list of all movies.

2. Get Single Movie

URL: GET /movies/:title

Description: Return data about a single movie by its title.

3. Get Genre Data

URL: GET /genres/:name

Description: Return data about a genre by its name.

4. Get Director Data

URL: GET /directors/:name

Description: Return data about a director by their name.
